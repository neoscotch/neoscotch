#### Fixing PATH

Try:

```javascript
PATH=/bin:/usr/bin:/usr/local/bin:${PATH}
export 
```
Alternatively, for "resetting" zsh, specify the complete path to the shell:

```javascript
exec /bin/zsh
```
or

```javascript
exec /usr/bin/zsh
```
